# Грамматические темы

Каждая тема хранится в отдельном JSON:

```text
data/grammar/grammar-1.json
```

## Структура

```json
{
  "id": "grammar-1",
  "number": 1,
  "title": "Verb to be: am, is, are",
  "subtitle": "Как сказать «я есть», «она есть», «они есть»",
  "status": "available",
  "linkedLessonId": "lesson-1",
  "notification": {
    "enabled": true,
    "version": 1
  },
  "explanation": [
    {
      "title": "Главное правило",
      "text": [
        "В английском предложении нужен глагол.",
        "После I используем am, после he, she, it — is, после we, you, they — are."
      ],
      "examples": [
        {"en": "I am a student.", "ru": "Я студент."},
        {"en": "She is at home.", "ru": "Она дома."}
      ],
      "table": {
        "headers": ["Кто?", "Форма", "Пример"],
        "rows": [
          ["I", "am", "I am happy."],
          ["he / she / it", "is", "She is happy."],
          ["we / you / they", "are", "They are happy."]
        ]
      }
    }
  ],
  "commonMistakes": [
    "Нельзя говорить I is.",
    "В утвердительном предложении нельзя пропускать am, is или are."
  ],
  "exercises": []
}
```

Для уровня A1 объяснения, инструкции, подсказки и сообщения проверки должны быть преимущественно на русском. Английский остаётся в изучаемых правилах, примерах и заданиях.

Добавляйте ровно пять заданий с постепенным усложнением:

1. самое лёгкое — выбор одной формы;
2. простое заполнение пропусков;
3. утверждения, отрицания и вопросы;
4. исправление ошибок;
5. самое сложное — самостоятельное преобразование или перевод.

Формат элементов упражнений совпадает с форматом вопросов домашней работы из `data/lessons/README.md`.
